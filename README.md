## 📌 Project Overview

The shift toward remote and hybrid work has transformed modern workplaces. This project analyzes how different work models (Remote, Hybrid, and Onsite) influence employee productivity, stress levels, mental wellbeing, work-life balance, and perceived organizational support.

The project combines **Excel datasets**, **SQL**, and **Power BI** to clean, analyze, and visualize employee wellbeing metrics, enabling data-driven workplace decisions.

---

## 🎯 Objectives

- Analyze employee productivity across different work models.
- Compare employee wellbeing metrics.
- Identify patterns in stress levels and mental health.
- Measure work-life balance across job roles.
- Build an interactive Power BI dashboard for business insights.
